## Summary

The team for this hackathon was: me, Christopher, and Henry (my previous hackathon team that competed in [Cal Hacks 2021](/index/experiences/competition_calgary_hacks_2021/).

This hackathon was interesting. We decided to compete in a track where we used DCP (Distributed Compute Protocol) a start up company in Kingston that had invented a way for you to harness the compute power of multiple distributed devices across the Internet using browser technology.

We couldn't think of a good project idea, since DCP does not let you install packages or access the Internet from within the workers.

Although we did not place, we still got interviews with the company since we were a good contender.
